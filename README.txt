== Description ==

This is a simple plugin which modify your woocommerce order table and adds a new column.

Modification of the order table to add a simple column which shows the ordered product and its quantity in front.
Without opening orders shop owner can view what product is ordered with quantity with respective their
respective name.

Format of item show in Modified column is simple like 
    "Product name x quantity"



== Installation ==

1. Upload `zwk-item-list.php` to the `/wp-content/plugins/` directory

== Frequently Asked Questions ==

= Does it render something on order table =

No it just add column name item

= Does it effect my orders too =

No it has no effect on order
